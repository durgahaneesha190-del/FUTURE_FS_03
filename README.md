# 🌐 Task 3 – Local Business Website & Live Pitch Project

## 📌 Project Title
Glow Beauty Salon – Professional Business Website

## 📖 Project Description
This project is a responsive business website developed for a local beauty salon. The website helps promote the business online by showcasing its services, gallery, customer testimonials, and contact information. It provides a modern user interface and improves the salon's digital presence.

---

## 🎯 Objective
To design and develop a professional website for a real local business and demonstrate how a website can help increase customer engagement and business growth.

---

## 🚀 Features

- Responsive Design
- Modern Landing Page
- About Section
- Services Section
- Image Gallery
- Customer Testimonials
- Contact Form
- Google Maps Integration
- Social Media Links
- Smooth Scrolling Navigation
- Mobile-Friendly Layout

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome

---

## ▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The website will open in your default web browser.
